# MERN Portfolio (Upgraded)

## Features
- Glassmorphism + gradient UI
- Responsive layout with animations
- Contact form that posts to Express backend and stores messages in MongoDB

## Quick start
1. Copy server/.env.example -> server/.env and set MONGO_URI.
2. Run server:
   ```
   cd server
   npm install
   npm run dev
   ```
3. Run client:
   ```
   cd client
   npm install
   npm start
   ```
